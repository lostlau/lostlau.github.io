Ocra: progressive learning

作者: Lost Blog 日期: 2023-06-17 阅读时间:1.5 分钟 字数:413 阅读量: -

小模型微调的几大挑战：

  1. Simple instructions with limited diversity
    之前提出了用Self-Instruct技术来更大的LLM自行扩充语料库，但是实证发现self-instruct得到预料在多样化和复杂度上面很所欠缺，不如人工标注提供的贡献度高。
  2. Task diversity and data scaling
    当前人工标注语料库主要是在创意内容生成和信息找寻这两个范畴比较多，这也导致最终训练出的模型更在意某些形式，而不是真正捕捉到“推理”的过程。
  3. Limited imitation signals
    缺少追踪推理过程的信号。
  4. Evaluation
    当前对小模型的表现评估方式不合理。

Preliminaries

  1. Instruction Tuning
    “指令微调”是一种技术用来让pre-trained model能够从<input, response>二元对中进行学习，即从指令中学习如何给出正当的回答。原始的指令微调通常用的是一些简练的<input, response>二元对，如下图：

    所以在小模型的微调训练中（通常是让大模型来“带”小模型学习，让大模型生成指令二元组，喂给小模型学习），在推理过程的理解学习中（更多的是对大模型回答的模仿，而非理解）有比较大的短板。
  2. System Instructions
    参照openai里chatgpt的chat completion里面最初设置的role = "system"的prompt，就是系统给出模型在回答时需模仿的角色画像。该方式可以较大的提高模型的安全性，比如在回答里什么是不能说的的，什么原则是需要被遵守的。
